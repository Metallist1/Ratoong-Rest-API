GET_FILTERED_RESORT
This API returns single resort with detailed filtered rating data over a period of time.
  • Required: id
    • This is ID assigned to every resort.
  • Optional: level
    • This filter will only include ratings from users that marked a specific experience level they have. Levels: none, little, some, expert
  • Optional: weeks
    • This filter will only include ratings from users that marked a specific amount of weeks per year that they are skiing. Weeks: rare, common, frequent
  • Optional: purpose
    • This filter will only include ratings from users that marked a specific purpose for skiing. Purpose: school, offpiste, party, relax, spa, shopping
  • Optional: snowboarder
    • This filter will only include ratings from users that selected snowboarding as their preferred way of skiing. Format: true / false.
  • Optional: age
    • This filter will only include ratings from users whose age is in the range. Format: From age - To age. Example : 10-17
  • Optional: skier
    • This filter will only include ratings from users that selected skiing as their preferred way of skiing. Format: true / false.
  • Optional: gender
    • This filter will only include ratings that are from specific gender. Genders: male, female, other.
  • Optional: country
    • This filter will only include ratings that are from specific country. Example: Denmark
  • Optional: fromDate
    • This filter will include out all ratings from a specific date. Format : Year-Month-Day. Example: 1970-07-15
  • Optional: toDate
    • This filter will include out all ratings until a specific date. Format : Year-Month-Day. Example: 1999-07-15
  • Required: Api key
    • Your API key. Claim your free API key in My Profile 
            {{getFilteredLocation| json}}
Link example: https://us-central1-final-exam-9302a.cloudfunctions.net/getFilteredResort?id=1
